# 🌍 Interactive World Map

An interactive web application that allows users to click anywhere on the world map to instantly explore country facts, COVID-19 statistics, and travel insights.

## 🚀 Features
- Interactive world map using Leaflet.js
- Real-time country data from REST Countries API
- Live COVID-19 statistics via disease.sh API
- Responsive modern UI
- Clean, readable vanilla JavaScript

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
- Leaflet.js
- OpenStreetMap
- Public APIs

## 📚 What This Demonstrates
- API integration
- Asynchronous JavaScript (fetch / async-await)
- Event handling
- Responsive design
- Real-world data visualization

## 📸 Preview
Clone the repo and open `index.html` in your browser.
